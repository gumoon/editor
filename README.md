## 常用命令

是否安装了 C 编译器：`cc --version`

是否安装了 make: `make -v`

编译：`make`

执行：`./editor`

在 Git 中，提交代码时使用的用户名和电子邮件地址是本地的配置，不是 GitHub 的用户名。
如果你只想在特定的仓库中生效，在该仓库目录下运行以下命令。

```
git config user.name "yourname"
git config user.email "youremail"
```
